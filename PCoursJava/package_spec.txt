Level package :

map.png : image représentant la map
level.xml : fichier xml qui contient ce que représente chaque couleur
			Par exemple : Wall = #0000; empty = #FFFF, YoannBlanc = #AB89

Un répértoire par Things, un répértoire contient les ressources de la Thing : images, script de comportement.
Exemple :
	YoannFrame1.png
	YoannFrame2.png
	YoannDead.png
	behavior.java
	
On le programme va compiler les .java, et créer un package qui sera réutilisé.
